# Style Guide (LadiPage Setup)

## Colors

- Primary dark: `#0B1F3B`
- Primary blue: `#0E5FD7`
- Accent cyan: `#14B8A6`
- Soft light text: `#E9F3FF`
- Gold highlight (headline effect): `#FFD713`

## Suggested Gradients

- Header background:
  `linear-gradient(90deg, #0B1F3B 0%, #0E5FD7 100%)`
- CTA buttons:
  `linear-gradient(90deg, #14B8A6 0%, #0EA5E9 100%)`
- FAQ title bars:
  `linear-gradient(90deg, #2563EB 0%, #0EA5E9 100%)`

## Typography

- Hero/Section titles: bold display style (Play/Playfair equivalent)
- Body text: clean sans-serif (Open Sans / Nunito style)
- Suggested hierarchy:
  - H1: 42-52 desktop, 28-34 mobile
  - H2: 30-36 desktop, 24-28 mobile
  - Body: 15-18 desktop, 14-16 mobile

## Spacing

- Section vertical padding: 60-90 px desktop, 36-56 px mobile
- Gap between major blocks: 28-40 px
- Form field height: 46-52 px
- Button height: 44-52 px

## Logo Usage

- Upload: `assets/aircerttechsafety-logo.svg`
- Header logo container:
  - fit: contain
  - align: center
  - no crop
- Footer logo container:
  - fit: contain
  - align: center
  - no crop

## Optional Decorative Images

- Header decoration: `assets/header-generated.webp`
- Footer decoration: `assets/footer-generated.webp`

Use decorative images as background with 30-50% overlay to keep text readable.
